## Contributing
This is a personal research project based on manual OSINT observation.

### Dataset Contributions
I'm currently not accepting direct contributions to the dataset itself.

The observation and documentation process is manual and maintained by the project owner to ensure methodological consistency.

### Methodology Feedback
Feedback on methodology is welcome and encouraged:

- **Discussions**: Open a [discussion](https://github.com/DarthJahus/telegram-fraud-network/discussions) for:
  - Methodology suggestions
  - Questions about the approach
  - General feedback

- **Pull Requests**: PRs are welcome for:
  - Documentation improvements
  - Typo fixes
  - Clarifications in method files

## Boundaries
Before contributing, please read:
- [Ethics](ethics.md) - ethical constraints
- [Scope](scope.md) - what is included/excluded
- [Method](method.md) - documentation principles

Thank you for your interest in improving this methodology.
